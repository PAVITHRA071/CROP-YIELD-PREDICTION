Crop Yield Prediction System

Description

This project predicts crop yield using Machine Learning. It uses different models to analyze data and give accurate predictions.

Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn

Models Used

* Linear Regression
* Decision Tree
* Random Forest

Features

* Data preprocessing
* Model training
* Prediction
* Graph visualization

Result

Random Forest gives the best accuracy among all models.
